Social Media Sentiment Analysis

NLP Machine Learning

Project Overview

This project focuses on analyzing social media text data to classify user sentiment into Positive, Negative, and Neutral categories using Natural Language Processing techniques.

Approach

  • Text cleaning and preprocessing
  • Feature extraction using TF-IDF
  • Model training and evaluation

Models & Tools

  • Python
  • NLTK
  • Scikit-learn
  • Logistic Regression & Naive Bayes

Results

Achieved approximately 85% accuracy in sentiment classification, enabling meaningful insights from social media data.
