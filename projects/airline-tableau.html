Airline Delay Analysis

Project Overview

This project analyzes airline flight delay patterns across different airlines, airports, and time periods to identify operational inefficiencies and trends.

Key Objectives

  • Identify airlines with the highest delays
  • Analyze airport-wise delay performance
  • Study seasonal and time-based delay trends

Tools & Technologies

  • Tableau
  • Excel
  • Data Cleaning & Aggregation
  • Exploratory Data Analysis
