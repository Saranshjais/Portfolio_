HungryBird

Web Application Map-Based

Project Overview

HungryBird is a location-based web application designed to help users discover nearby street food vendors using interactive maps and geolocation data.

Key Features

  • Real-time street food discovery
  • Interactive map with vendor locations
  • User-friendly interface for local exploration

Tech Stack

  • Python
  • Flask
  • Leaflet.js
  • Geolocation APIs
