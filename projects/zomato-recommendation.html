Zomato Recommendation System

Machine Learning Recommendation System

Project Overview

This project implements a restaurant recommendation system that suggests relevant dining options to users based on their preferences, ratings, and similarity between restaurants.

Problem Statement

With thousands of restaurants available, users often struggle to find places that match their taste. This system aims to simplify discovery through personalized recommendations.

Approach

  • Exploratory data analysis on Zomato dataset
  • Feature engineering on cuisines, ratings, and location
  • Similarity calculation using cosine similarity

Tools & Technologies

  • Python
  • Pandas & NumPy
  • Scikit-learn
  • Cosine Similarity

Outcome

The model provides accurate and relevant restaurant suggestions, improving user experience and decision-making.
